# Turn Order Manager
DM Tools

### Features
- Initiative tracker
- Turn timer
- Turn history
- Dice roller
  - Example 1. `5d20 + 3` 5x d20s and add 3
  - Example 2. `3d7 - 2` Roll 3x d7 and minus 2
  - Example 3. `1ad20` Roll 1x d20 with advantage
  - Example 4. `3dd4` Roll 3x d4 with disadvantage
  - Example 5. `1d6 + 2ad3 + 3dd9 + 2 + 1`
